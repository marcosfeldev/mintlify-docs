## Criando um usuário no banco de dados
1. Clique no user que está em cima do lado esquerdo.
2. Clique em **Adicionar**.
3. Preencha os campos e marque a opção **Privilégios globais** conforme a imagem abaixo.
3. Clique em salvar e pronto, **usuário criado**.

## Conectando o BOT

Agora que você possui um usuário no banco de dados, vamos conectar ao BOT.

1. Digite o comando `/config-database` no seu **BOT**.
2. Preencha o formulário aberto com as informações do banco de dados que você criou no passo anterior.
3. Preencha o nome da tabela do seu banco de dados `(veja o próximo tópico sobre onde encontrar)` e clique em Enviar.
