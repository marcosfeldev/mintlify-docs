## Pegue o ID de qualquer canal no servidor
Para localizar o ID de um canal no Discord, é necessário ativar antes o Modo Desenvolvedor na sua conta. Siga os passos abaixo:
1. Acesse a **configurações da sua conta** no Discord.
2. Clique em **Avançado**.
3. Habilite o **Modo Desenvolvedor**.

<Frame>
  ![Conseguir ID do canal](/images/shared/developer_mode/developermod-1.jpg)
</Frame>

Depois de ativar o Modo Desenvolvedor, siga estas etapas para copiar o ID de um cargo:

1. Entre no seu servidor no Discord.
2. Localize o canal desejado.
3. Clique com o botão direito sobre ele.
4. Selecione a opção **Copiar ID**.

  ![Conseguir ID do canal](/images/shared/get_channel/channel-1.jpg)
</Frame>
