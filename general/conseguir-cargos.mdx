## Pegue o ID de qualquer cargo no servidor
Para localizar o ID de um cargo no Discord, é necessário ativar antes o Modo Desenvolvedor na sua conta. Siga os passos abaixo:
1. Acesse a **configurações da sua conta** no Discord.
2. Clique em **Avançado**.
3. Habilite o **Modo Desenvolvedor**.

<Frame>
  ![Conseguir ID Cargo](/images/shared/developer_mode/developermod-1.jpg)
</Frame>

Depois de ativar o Modo Desenvolvedor, siga estas etapas para copiar o ID de um cargo:

1. Entre no seu servidor no Discord.
2. Abra o menu de **Configurações do servidor**.
3. Vá até a seção **Cargos**.
4. Localize o cargo desejado.
5. Clique com o botão direito sobre ele.
6. Selecione a opção **Copiar ID**.

<Frame>
  ![Conseguir ID Cargo](/images/shared/developer_mode/developermod-2.jpg)
</Frame>
